FormulaDrift

FormulaNight is een race game die in de nacht afspeeld. Ik heb hier vooral gewerkt aan opslaan van bijbouden van tijden en laps (school jaar 2)

Project Info

Dit project is gemaakt voor school in jaar 2 en we moesten een night racer maken in 4 weken. De bedoeling van dit project was eerst dat het een open word race game zou worden waarbij er meerdere races beschikbaar zijn om op te racen. aan de art kan hebben ze dat niet kunnen redden maar de backend zoals opslaan van scores en de Game/Race managers werken groot en deels om open world aan te kunnen.

Tijdens dit project heb ik gewerkt aan.

    UI functionalitijd

    Eindsherm UI

    Er komt veel te staan op het eind scherm dus heb ik er voor gekozen om alle checkpoint/segment tijden op een apart tabje te zetten met elke checkpoint een eigen kollom, dit zodat alles goed overzichterlijk is.

    • Code

    GameUI

    Na elke checkpoint krijg je aan de linker kant van het beeldscherm ook te zien hoe lang je over de lap hebt gedaan en waar in de lijst de tijd staat. Dit is voor 2 seconden te zien na het halen van een checkpoint.

    • Code

    GameMamanger

    LapManager

    De lapManager houd de tijd en aantal laps bij van de race die bezig is en slaat daarna alles op naar de game saver die alles omzet naar een save file. De lapManager word alleen gebruikt als er een race is gestart en elke race heeft zijn eigen lapManager

    • Code

    GlobalLapManager

    De GlobalLapManager word gebruikt om dingen die altijd actief zijn aan te passen zoals de text voor welke lap bezig is. De GlobalLapManager zorgt er voor dat er genoegt opslag plekken zijn voor alle data en laad het in laden als er een safe file bestaat. Het zogt voor het respawnen van de player en stopt de player van rijden wanneer dat nodig is zoals tijdens het aftellen op de start. Als er maar 1 race is om te spelen dan zogt de GlobalLapManager er voor dat die race automatisch word gestart.

    • Code

    Checkpoints

    De checkpoint slaat de player positie op en geeft de opdracht aan de LapManager om tijden op de slaan en te kijken of er een lap voorbij is.

    • Code

    GameSaving

    DataCenter

    De DataCenter slaat de game op in een aparte file en laad die weer in als de game word gestart en er een safe file bestaat. Voor het opslaan maak ik gebruik van een BinaryFormatter die een class met variables omzet naar een file.

    Navigatie Pijltje

    Ik heb een navigaite pijltje gemaakt die boven de auto hangt zodat de player kan zien waar de volgende checkpoint zit. Nu de game een loop is, is het niet zo zeer nodig maar met het eerste idee van een open world game is het wel handig om te zien waar je heen moet.
