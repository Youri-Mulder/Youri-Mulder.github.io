DualDefenders

DualDefenders is een tower defence game gemaakt door 2 developers. in deze TW moet je niet alleen je eigen base beschermen maar moet je ook enemies sturen naar de Ai tegenstander

Tijdens dit project heb ik gewerkt aan.

  • Camera movement
  • TegenstanderAI
  • Enemies plaatsen
    •

    CameraMovement

    Om te kunnen wisselen van jouw velt naar die van de player heb ik een camera gemaakt die van een start vector naar een eindvector gaat doormiddel van een slerp. Waar de camera zich bevind heeft ook effect op welke kant van de map je wel en niet kan horen.

    • Code

    Enemies

    We moesten voor dit project voor de movement een pathfinding systeem van een docent gebruiken. ik en mijn team mate hebben gekozen voor de waypoints pathfinding. de rest van de enemies heb ik zelf gemaakt. De enemies hebben scriptable objects om heel makkelijk verschillende varianten te maken.

    Enemy Base

    In de ai base worden alle values ingestelt is staat de code die word gebruikt als de enemie zijn eindpunt heeft berrijkt.

    Shooting Enemy

    Voor de shooting enemy gebuik ik een overlapssphere om te kijken of er een tower in de enemie range zit. er word vervolgens een kogel gespawnt en gedraait richting de eerste de beste tower. als laatst word er nog een coroutine gestart voor cooldown.

    • Code

    Tegenstander AI

    De tegestander AI heeft meerdere states en zo de mogelijkheid om towers te plaatsen en upgraden. met de chosing state word er gekeken wat er in de map nodig is. als er genoeg towers zijn over de map dan gaat de ai of meer bij bouwen of towers upgraden en dit is allemaal instelbaar. zodat de ai niet towers en upgrades gaat spammen is er ook een idle state.

    • Code

    Enemies plaatsen

    De enemies kunnen geplaatst worden op spawn tiles. Via de spawn manager word er dan een enemie gespawnt op die tile. Van de spawn tile word een tag gepakt dat gelijk staat aan een pad dat de enemy moet gaan volgen.
