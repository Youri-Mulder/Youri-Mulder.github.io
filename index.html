Over mij

Ik ben Youri Mulder, jaar oud, en ik ben een gepassioneerde game developer. Programmeren op zichzelf vind ik leuk om te doen, of dat nu in PHP, C# of een andere taal is, maar het programmeren binnen game development spreekt mij het meest aan. Dit komt vooral doordat games interactief zijn en de logica die je schrijft sterk afhankelijk is van het type game dat je maakt geen enkel systeem voelt hetzelfde.

Mijn interesse in game development begon met GameMaker Studio, waar ik voor het eerst in aanraking kwam met visual scripting via drag and drop codeblokken. Later stapte ik over naar Roblox Studio, waar ik mijn eerste ervaring met echte code opdeed via tutorials. Daar bleef het niet bij, ik paste code aan, combineerde scripts en experimenteerde om beter te begrijpen hoe alles samenwerkte. Deze nieuwsgierigheid nam ik mee toen ik Unity ontdekte, wat uiteindelijk leidde tot mijn start bij de opleiding Game Development aan het Grafisch Lyceum Utrecht, waar ik nu in mijn derde leerjaar zit.

Na mijn opleiding wil ik mij verder blijven ontwikkelen binnen game development en mij blijven verdiepen in het bouwen van sterke gameplay en game systemen.
