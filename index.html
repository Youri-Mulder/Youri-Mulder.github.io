Over mij

Ik ben Youri Mulder en ik vind het geweldig om games te maken. Het maakt mij over het algemeen niet heel veel uit aan wat voor soort game ik werk Waar ik de plezier uithaal is een systeem bedenken om een feature te maken en dat vervolgens ook werkent te krijgen. Ik volg nu de opleiding game development op het GLU (Grafish lyseum Utrecht) en zit in jaar 2.

Naast het ontwikkelen van games vind ik het uiteraard ook heel leuk om zelf te gamen. Zelf speel ik voornaamelijk veel rythem games maar shooters speel ik ook best veel. Naast het binnen zitten en bezig zijn met games ben ik ook regelmatig buiten met vrienden.

Wat mij de inspiratie heeft gegeven om game developer te worden was de end cutscene van ark extintion. Het leek me zo vet om een game te maken met een heel verhaal er achter en daar onderdeel van te zijn. Ook de game developer youtuber Dani heeft mij inspiratie gegeven toen game development voor mij een optie werd.

Mijn eerste ervaren met game development waren met Gamemaker Studio en Roblox Studio. In Gamemaker Studio had ik toen een player werkend gekregen met drag boxes en in Roblox gebruike ik voor het eerst code met hulp van een tutorial video. Later ben ik gaan rondkijken in Unity en een jaar later ben ik gestart op het GLU.

Skills

Unity -

Ik kan goed over weg met Unity en kan overgroot deel van alle components gebruiken. Ik kan games maken in 2D, 3D en VR. Ik kan werken met veel gebruike packages zoals bijvoorbeeld cinemaschine en post prosessing. Ik snap hoe occlusion werkt en hoe ik dat moet toepassen. Ook begin ik nu steeds vaker scriptable objects in projecten toe te voegen. En ik snap hoe de nieuwe unity input system werkt voor groot en deels.

C# -

Ik kan van veel unity ingebouwde functies zoals rotatetowards gebruik maken. Ik snap hoe ik van scripts kan over erven en intherfaces / structs kan toevoegen. Ik ken de S.O.L.I.T prensiepes en kan mijn code met die methode structureren. Ook voeg ik bij mijn code commands toe voor uitleg.

Git -

Ik gebruik als mijn main git client fork verder heb ik ook gewerkt met github desktop en kan ik werken met git in de command line. Ik kan snel overweg met nieuwe git clients mocht dat nodig zijn. Ik snap hoe ik kan werken met branches, van commits kan verwisselen en hoe ik merge error oplos.
