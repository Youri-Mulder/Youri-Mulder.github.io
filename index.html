Over mij

Ik ben Youri Mulder en ik vind het geweldig om games te maken. Het maakt mij over het algemeen niet heel veel uit aan wat voor soort game ik werk Waar ik de plezier uithaal is een systeem bedenken om een feature te maken en dat vervolgens ook werkent te krijgen. Ik volg nu de opleiding game development op het GLU (Grafish lyseum Utrecht) en zit in jaar 3.

Naast het ontwikkelen van games vind ik het uiteraard ook heel leuk om zelf te gamen. Zelf speel ik voornaamelijk veel rythem games maar shooters speel ik ook best veel. Naast het binnen zitten en bezig zijn met games ben ik ook regelmatig buiten met vrienden.

Wat mij de inspiratie heeft gegeven om game developer te worden was de end cutscene van ark extintion. Het leek me zo vet om een game te maken met een heel verhaal er achter en daar onderdeel van te zijn. Ook de game developer youtuber Dani heeft mij inspiratie gegeven toen game development voor mij een optie werd.

Mijn eerste ervaren met game development waren met Gamemaker Studio en Roblox Studio. In Gamemaker Studio had ik toen een player werkend gekregen met drag boxes en in Roblox gebruike ik voor het eerst code met hulp van een tutorial video. Later ben ik gaan rondkijken in Unity en een jaar later ben ik gestart op het GLU.
