Over mij

Ik ben Youri Mulder 19 jaar oud en ik vind het geweldig om games te maken. Het maakt me over het algemeen niet veel uit aan wat voor soort game ik werk. Waar ik plezier uithaal is het bedenken van een systeem om een feature te maken en dat vervolgens ook werkend te krijgen. Het leukste om aan te werken vind ik toch wel gameplay maar met de lessen nu over PHP vind ik backend ook best leuk. Ik volg nu de opleiding Game Development aan het GLU (Grafisch Lyceum Utrecht) en zit in het derde jaar.

Naast het ontwikkelen van games vind ik het uiteraard ook heel leuk om zelf te gamen. Ik speel voornamelijk veel rhythm games maar shooters speel ik ook best veel. Naast het binnen zitten en bezig zijn met games ben ik ook regelmatig buiten met vrienden.

Wat mij de inspiratie heeft gegeven om game developer te worden was de end cutscene van Ark Extinction. Het leek me zo vet om een game te maken met een heel verhaal erachter en daar onderdeel van te zijn. Ook de game developer YouTuber Dani heeft mij inspiratie gegeven toen game development voor mij een optie werd.

Mijn eerste ervaringen met game development waren met GameMaker Studio en Roblox Studio. In GameMaker Studio had ik toen een player werkend gekregen met drag boxes en in Roblox gebruikte ik voor het eerst code met hulp van een tutorialvideo. Later ben ik gaan rondkijken in Unity en ongeveer een jaar later ben ik gestart op het GLU.
